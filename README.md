# &lt;/VDB-CLOCK&gt;
 A simple note taking TUI app.

# </How to install?>

## //Linux
First clone the repository:
```bash
git clone https://github.com/TibeDev/vdb-notes.git
cd vdb-notes
```

Then install it via the install script:

```bash
chmod +x install.sh
./install.sh
```

After that you can run "notes" in the terminal.

```bash
notes
```
